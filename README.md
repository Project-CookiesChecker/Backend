🛠️ สิ่งที่ต้องมีก่อน (Prerequisites)
ก่อนเริ่มรันโปรเจกต์ ต้องติดตั้งโปรแกรมเหล่านี้ก่อน:

🚀 ขั้นตอนการติดตั้ง (Installation)
ติดตั้ง Python Library
เปิด Terminal (CMD หรือ PowerShell) ในโฟลเดอร์โปรเจกต์ แล้วรันคำสั่งนี้: !!!อย่าลืมทำใน ven10 นะ!!!

pip install -r requirements.txt

นำโฟลเดอร์ hf_cookie_model_balanced ไปวางไว้ในโฟลเดอร์โปรเจกต์ CookiesChecker

รัน Python Server
เปิด Terminal ในโฟลเดอร์งาน

พิมพ์คำสั่ง:
python api_server.py

ขั้นตอนลง Extension ใน Chrome
เปิด Chrome แล้วพิมพ์ช่อง URL ว่า chrome://extensions/

เปิดสวิตช์ Developer mode (มุมขวาบน)

กดปุ่ม Load unpacked (มุมซ้ายบน)

เลือกโฟลเดอร์ Extension ใน CookiesChecker
